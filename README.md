# MyChatsApp

## Overview

MyChatsApp is a Java-based messaging application developed using Maven and NetBeans. The application allows users to register, log in, send messages, store messages in JSON format, and validate recipient information.

## Features

* User registration
* User login validation
* Message creation
* Message hashing
* Message storage using JSON
* Recipient number validation
* JUnit testing
* GitHub Actions CI workflow

## Technologies Used

* Java 17
* Maven
* JUnit 5
* org.json Library
* GitHub Actions
